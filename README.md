# STM32F429i Discovery Example

Example STM32F4 project adapted from the [blinky] example and adapted for
the [STM32F429i Discovery][1] board.

## Requirements

* [stlink](https://github.com/texane/stlink#readme)
* arm-none-eabi toolchain

[blinky]: https://github.com/Malkavian/tuts/tree/master/stm/blinky
[1]: http://www.st.com/web/catalog/tools/FM116/SC959/SS1532/PF259090
